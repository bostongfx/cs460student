RO.ME '3 DREAMS OF BLACK'

Description: 3 Dreams of Black was created in 2011 by Google Data Arts Team. I think its impact was big ( at its time , 2011 !) because it proved 3D graphics and interactive stories could run in a browser without plugins, and it pushed more developers to use WebGL for creative websites. RO.ME is a very cool webgl demo!

Technology: WebGL, HTML5, JavaScript, Three.js, CSS3, and maybe some API for audio and video.

Comparison: No-Fishing website No-Fishing uses WebGL and Three.js too, but it is not as interactive as RO.ME. It is more like a 3D art gallery, while RO.ME is more like an interactive story. No-Fishing's main focus is to collect donations for marine conservation, while RO.ME is more like an art project. New Edit: At the very bottom of the page, I added two models (fox and horse) that I found on mrdoob's website.

Significance (AI): I asked ChatGPT why RO.ME was important. It said RO.ME showed that WebGL and HTML5 could create interactive 3D stories in the browser without plugins, and it inspired more creative web graphics projects.
