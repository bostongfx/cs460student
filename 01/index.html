RO.ME is an incredible WebGL demo!. This WebGL website is an insight into 3 dreams, the dreams have been visualized as an interactive 3D world. It's like a music video made for the browser. This really opened the door for creativity when it comes to web design fusing audio, video, real-time rendering etc.
