RO.ME "3 DREAMS OF BLACK"

RO.ME: 3 Dreams of Black came out in 2011, created by Chris Milk and Google Creative Lab. It was one of the first interactive films to run entirely in a web browser using WebGL. The demo showed that web browsers could handle 3D graphics and interactive storytelling.

Technology: WebGL for rendering 3D graphics, Three.js JavaScript library, HTML5 , JavaScript, Google Chrome.

Comparison: WLDLGHT is an interactive WebGL experience that focuses on exploring light and shadow in a 3D environment. Unlike RO.ME, which tells a cinematic story, WLDLGHT emphasizes visual experimentation and user interaction rather than narrative.

Significance (AI): RO.ME was important because it demonstrated that fully interactive, cinematic 3D experiences could run directly in a web browser using WebGL, without any plugins like Flash, highlighting the web's potential for immersive storytelling.
