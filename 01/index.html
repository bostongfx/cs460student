RO.ME "3 DREAMS OF BLACK"

RO.ME is a an immersive experience that utilizes WEBGL to take the user through a 2D and 3D film. It tells the story of a song through a three phase plan. The first phase begins with a couple 2D images that gradually transforms as well as cuts to new images. The second phase takes the user through a 3D world in which the user uses the mouse to inspect the world as well as direct a stream of graphics through a predetermined track. The third phase retains similar elements from the second phase but also expands on it by allowing the user to direct themselves through the world.

Technology: RO.ME uses javascript and HTML to utilize the GPU in your computer to render images in your browser.

Comparison: Try Me Lusion is another example of browser graphical rendering. While RO.ME takes you through a modeled experience, Lusion is based more in user web-based interaction. As you scroll through the website, the user is met with interaction canvases that demonstrate the power of WEBGL just as well.
